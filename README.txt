Simulador de carrito de compras hecho con HTML, CSS y JavaScript
 - El proyecto funciona como una single page aplication.
 - Se puede filtrar los productos por categoría. 
 - El contenido de los productos se genera utilizando JavaScript, leyendo los datos del archivo productos.json
 - Desde la pagina "carrito" se visualizacada producto agregado, nombre, precio, cantidad y subtotal. Podemos eliminar una unidad del producto en caso de requerirlo haciedno click en el boton X.
 - Al final se peude ver el total de la compra y tenemos la opción "Vaciar el carrito" con un solo boton.
 - Se utiliza la libreria SweetAlert2 en las confirmaciones del carrito.
