Simulador de carrito de compras hecho con HTML, CSS y JavaScript
- Se puede agregar la cantidad de productos deseados desde varias páginas. 
- Se almacenan en localStorage
- Desde carrito.html se visualiza por cada producto agregado, el nombre del producto, el precio, la cantidad y el subtotal. Podemos eliminar una unidad del producto haciendo click en la X.
- Al final se puede ver el total de la compra y existe la opción de vaciar el carrito con un solo botón.
